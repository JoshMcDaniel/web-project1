Hyper Text Markup Language 5 (HTML5)

HTML5 is a powerful tool used to create websites that are more functional while also making them easier to code for the web developer. One powerful tool used in HTML5 is an Application Programming Interface (API) which allows a website to utilize a program or operating system to perform useful tasks. Below you will see the Geolocation API which uses your devices GPS to display your current location on s map. This feature was previously only possible with downloaded applications. This is just one of many new features added with HTML5.

Click the button to get your location.

Cascading Style Sheet 3 (CSS3)

CSS3 adds many amazing syle features to websites that were once only possible with complex JavaScript code. Some new features include rounded corners for boxes or images, shadowing (as seen in this webpage text), and animation just to name a few. Below you will see the animation feature in action which will alter the size of an image as your cursor hovers over it. Give it a try!
