Front End (Client-Side)

The front end of web development consists of creating the look and feel of portion of the website that the client will use and interact with. This is where the term "client-side" comes from. Typically, when developing the front end of a website, the developer will use a combination of HTML, CSS and JavaScript to design the content, layout and interactivity. The goal of a front end developer is to create an appealing webpage that is both easy to use and functional for the needs of the client. Click on the logos below to learn more!
