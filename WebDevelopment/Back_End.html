Back End (Server-Side)

The back end of a website is the how the client interacts with a server and database to send, recieve and process information. This information is stored on a server which is where the term "server-side" comes from. Different languages are used on the back end of development such as:

  • Ruby
  • PHP
  • Python

How does it all tie together?

These languages, and many others, are used to connect the client to a database on a server to store information. The goal of a back end developer is to create a functional server and database that connects seamlessly to the front end of a website. Click on the logos below to learn more!
